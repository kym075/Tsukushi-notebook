# つくしノート (Tsukushi Notebook) 📚✨

「つくしノート」は、勉強メモを整理し、装飾し、さらに **Google Gemini API** を使った自動3択クイズで理解度を確認できる、学習特化型のデスクトップノートアプリケーションです。
CustomTkinter を使用した美しくモダンなダーク/ライトモード対応のUIを備えています。

---

## ✨ 主な機能

* **🎨 リアルタイム文字装飾 (リッチテキスト)**
  文字サイズ（12〜32pt）や、上品で読みやすい6色のフォントカラーパレットを切り替えながら執筆できます。
* **📷 画像のインライン挿入**
  ノートの中に学習用の図や写真を直接挿入し、テキストと一緒に表示・保存できます。
* **🌙 スマートなダーク / ライトモード**
  目に優しいダークモードと、明るいライトモードを瞬時に切り替え可能。文字色もテーマに合わせて自動追従します。
* **🧠 AI理解度クイズ機能 (Google Gemini 連携)**
  ノートの内容をAIが自動で読み取り、あなただけの3択問題を生成します！
  * **APIキー設定時**: 本物のAI先生が、ノートの重要箇所から毎回異なる切り口でクイズを生成します。
  * **オフライン/未設定時**: メモ内の箇条書き（■語句：説明）を高度に自己解析してクイズを作る「モックモード」も搭載。
* **🔄 安心の自動保存**
  キー入力から1秒後に自動でデータを保存するため、保存し忘れの心配がありません。

## 🛠 セットアップ方法

### 1. 必要な環境
* **Python 3.8 以上**

### 2. インストール手順

リポジトリをクローンまたはダウンロードし、プロジェクトのルートディレクトリで以下のコマンドを実行して必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

### 3. 初期データの作成（任意）
初回起動時にサンプルデータから始めたい場合は、`notes.json.sample` を `notes.json` にコピーしてください。

```bash
# Windows (PowerShell)
copy notes.json.sample notes.json

# macOS / Linux
cp notes.json.sample notes.json
```
*(コピーしない場合でも、アプリ起動時に空のデータベースが自動作成されます)*

---

## 🚀 実行方法

以下のコマンドを実行してアプリケーションを起動します。

```bash
python main.py
```

---

## 💡 AIクイズ（Gemini API）の利用手順

1. アプリ左下の「⚙ Gemini API 設定」ボタンをクリックします。
2. ご自身の [Google AI Studio](https://aistudio.google.com/) 等から取得した Gemini API キーを入力します。
3. ノートを記述し、右上の「📝 AIクイズ」ボタンを押すことで、AIが自動でノートからクイズを生成します。
   *(※APIキーはローカルの `notes.json` にのみ安全に保存され、外部に送信されることはありません)*
